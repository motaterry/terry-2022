# API Server

Express API server for job posting trends.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Environment Variables

- `DB_URL`: Path to SQLite database (default: `./db/jobs.db`)
- `API_PORT`: Server port (default: `3000`)
- `NODE_ENV`: Environment mode (`development` or `production`)
